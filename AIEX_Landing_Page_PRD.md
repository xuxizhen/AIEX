## 1. 页面目标

### 定位
AIEX 品牌营销着陆页，面向潜在用户（加密新手 + 有经验交易者），传达产品核心价值，引导下载/注册。

### 核心转化目标
- **主 CTA**: 下载 App / 开始交易
- **次 CTA**: 查看定价方案 / 了解更多

### 受众画像
| 用户类型 | 关注点 | 期望触达 |
|---------|--------|---------|
| 加密新手 | 简单易用、AI 帮忙赚钱、低门槛 | Hero + Nova AI + 定价 Free 档 |
| 有经验交易者 | 高级策略、低延迟、专业工具 | 交易界面 + 跟单广场 + ELITE/MAX 档 |
| 投资人/合作伙伴 | 差异化、商业模式、市场定位 | 全局 overview + 定价模型 |

## 2. 页面结构 (11 个 Section)

## 3. 视觉设计规范

### 配色方案
```
主色: #00e5ff (Electric Cyan)
辅助色: #ffd700 (Gold)
背景: #0a0e17 → #0f1923 (深色渐变)
卡片背景: rgba(255,255,255,0.03) (毛玻璃)
文字主色: #f0f4f8
文字次色: rgba(240,244,248,0.6)
成功色: #00e676 (Green)
危险色: #ff1744 (Red)
```

### 字体
- 英文标题: DM Sans (Display, Bold 700)
- 中文正文: Noto Sans SC (400/500/600)
- 数据/代码: JetBrains Mono

### 设计语言
- **Glassmorphism**: 毛玻璃卡片 + 半透明边框
- **粒子动画**: Hero 背景 Canvas 粒子连线
- **渐变光晕**: 品牌色径向渐变装饰
- **暗色主题**: 深海蓝底色，符合加密行业审美

### 响应式断点
| 断点 | 适配 |
|------|------|
| ≥1024px | 桌面端 — 并排布局 |
| 768-1023px | 平板 — 2列网格 |
| <768px | 手机 — 单列堆叠 |

## 4. 交互细节

### 动画
- **页面滚动**: Section 元素滚入视口时淡入上移 (IntersectionObserver)
- **粒子背景**: Canvas requestAnimationFrame 循环，粒子缓慢漂移 + 近距连线
- **悬停效果**: 卡片 translateY(-8px) + box-shadow 增强
- **CTA 按钮**: hover 时发光效果 (box-shadow 扩散)

### 导航交互
- 点击导航链接 → smooth scroll 到对应 Section
- 滚动时导航背景从透明变为毛玻璃
- 移动端: 汉堡菜单展开/收起

### 手机 Mockup
- 静态展示，内含实际 UI 内容 (非截图)
- HTML/CSS 构建的真实界面元素
- 展示 Nova 对话、跟单卡片、交易面板的关键界面

## 5. 技术实现

### 技术约束
- **单文件 HTML**: 所有 CSS/JS 内联，无外部依赖
- **Google Fonts**: DM Sans、Noto Sans SC、JetBrains Mono (唯一外部引用)
- **零框架**: 原生 HTML5 + CSS3 + JavaScript
- **性能**: 首屏 < 2s，总文件大小 < 200KB

### SEO 要素
- 语义化 HTML 标签 (header, main, section, footer)
- Meta description 和 viewport 设置
- 结构化标题层级 (h1 → h2 → h3)
