# AIEX Web Prototype — 产品需求与设计详述

**版本**: 1.0
**日期**: 2026-03-16
**文件**: `AIEX_Web_Prototype.html`
**状态**: 已完成

---

## 1. 产品目标

### 定位
AIEX Web 版是面向桌面浏览器的高保真可交互原型，**先于 APP 发布**，用于快速验证产品想法、团队评审和投资人展示。Web 版与 APP 功能 **1:1 对齐**，覆盖 AIEX 4 个页面 + Nova AI 右侧面板 + 订阅体系的核心交互流程。

### 核心 MVP 范围
| 模块 | 覆盖内容 | 交互程度 |
|------|---------|---------|
| 智能首页 | 资产面板、Nova 简报、快捷操作、行情表格、AI 洞察 | 完整 |
| 交易页 | 3 栏经典布局：K线图、盘口、下单表单 | 完整 |
| 跟单广场 | 交易员/AI 策略、免费/付费分层、订阅引导 | 完整 |
| 我的 | 用户信息、订阅管理、设置菜单 | 完整 |
| Nova AI 面板 | 右侧可收起面板、多轮对话、行动卡片、快捷提问 | 演示级 |
| 订阅弹窗 | 4 档方案居中弹窗展示 | 完整 |

### 技术约束
- **单文件 HTML**: CSS/JS 全部内联，自包含
- **外部依赖**: 仅 Google Fonts (DM Sans, JetBrains Mono, Noto Sans SC)
- **桌面优先**: min-width 1024px，max-width ~1920px 居中
- **无 Canvas / WebGL**: 纯 CSS + SVG
- **总文件大小**: < 150KB

## 2. 全局信息架构

### 顶部导航栏

```
┌──────────────────────────────────────────────────────────────────────────┐
│ [AIEX Logo]  智能首页 | 交易 | 跟单广场 | 我的    [Nova AI ✦] [🔔] [👤] │
└──────────────────────────────────────────────────────────────────────────┘
```

- **高度**: 60px
- **背景**: `--bg-secondary` (#111827) + `backdrop-filter: blur(20px)` + 底部 1px 边框 `rgba(255,255,255,0.06)`
- **固定定位**: `position: fixed; top: 0; z-index: 100`
- **左侧**: AIEX Logo (SVG，含 Cyan 渐变光效)，左边距 24px
- **中部**: 4 个页面链接，DM Sans 14px 500，间距 32px
  - Active 状态: Cyan 色文字 + 底部 2px Cyan 指示线
  - Hover 状态: `rgba(0,229,255,0.15)` 背景 + Cyan 文字
  - 非活跃: `--text-secondary` 色
- **右侧**: Nova AI 按钮 + 通知铃铛 + 用户头像，右边距 24px
  - Nova AI 按钮: 金色渐变背景圆角按钮，"✦ Nova" 文字，点击展开/收起右侧面板
  - 通知铃铛: 24px 图标，右上角红色小圆点 (未读计数)
  - 用户头像: 32px 圆形，Cyan→紫色渐变，字母 "M"

### Nova 右侧面板

**3 种状态**:

| 状态 | 呈现 | 触发 |
|------|------|------|
| 展开 | 320px 宽右侧面板，完整聊天界面 | 点击顶栏 Nova 按钮 / 浮动按钮 |
| 收起 | 浮动金色按钮，固定在右下角 | 点击面板收起按钮 (→) |
| 上下文 Chips | 面板底部显示当前页面相关快捷标签 | 自动根据页面切换 |

### 页面内容区

- **顶部偏移**: `padding-top: 60px` (为固定导航栏留出空间)
- **Nova 展开时**: 右侧留出 320px 空间，内容区自适应收缩
- **Nova 收起时**: 内容区占满全宽
- **过渡动画**: `margin-right 300ms ease`

### 层级体系

| 层级 | 元素 | z-index |
|------|------|---------|
| 1 | 页面内容 | 默认 |
| 2 | Nova 浮动按钮 | 70 |
| 3 | 顶部导航栏 | 100 |
| 4 | Nova 右侧面板 | 150 |
| 5 | 订阅方案弹窗 | 200 |
| 6 | 弹窗遮罩层 | 199 |

### 响应式策略

- **最小宽度**: 1024px (低于此宽度显示横向滚动条)
- **最大宽度**: 1920px (超出部分居中，两侧留白)
- **内容区**: max-width 1440px 居中 (Nova 收起时)
- **内容区**: max-width 1120px (Nova 展开时，1440 - 320)
