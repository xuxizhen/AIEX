## 1. 原型目标

### 定位
高保真可交互 App 原型，用于团队评审、投资人展示和开发参考。完整覆盖 AIEX 4 个 Tab 页面 + Nova AI 助手 + 订阅体系的核心交互流程。

| 模块 | 覆盖内容 | 交互程度 |
|------|---------|---------|
| 智能首页 | 资产面板、Nova 简报、快捷操作、行情、AI 洞察 | 完整 |
| 交易页 | K线图、AI 分析、下单表单 | 完整 |
| 跟单广场 | 交易员/AI 策略、免费/付费分层、订阅引导 | 完整 |

## 9. 视觉设计规范

### 配色 (CSS Variables)
```css
--bg-primary: #0a0e1a       /* 主背景 */
--bg-secondary: #111827     /* 次级背景 */
--bg-glass: rgba(255,255,255,0.04)  /* 毛玻璃 */
--cyan: #00e5ff             /* 品牌主色 */
--gold: #ffd700             /* 高级/Nova */
--green: #00e676            /* 涨/成功 */
--red: #ff1744              /* 跌/风险 */
--text-primary: #f0f4f8     /* 主文字 */
--text-secondary: rgba(240,244,248,0.6)  /* 次文字 */
--text-tertiary: rgba(240,244,248,0.35)  /* 弱文字 */
```

### 字体体系
| 用途 | 字体 | 字号 | 字重 |
|------|------|------|------|
| 页面标题 | DM Sans | 22px | 700 |
| 交易对名 | DM Sans | 14-20px | 600-700 |
| 数字/金额 | JetBrains Mono | 13-32px | 500-600 |
| 正文 | Noto Sans SC | 12-14px | 400-500 |
| 标签/Badge | DM Sans / Noto Sans SC | 9-11px | 600-700 |

### 圆角体系
```css
--radius: 16px      /* 大卡片 */
--radius-sm: 10px   /* 小卡片/输入框 */
--radius-xs: 6px    /* Badge/按钮 */
```

### 卡片设计语言
- **普通卡片**: `bg-glass` + 1px `border` + `backdrop-filter: blur(10px)`
- **AI 策略卡片**: Cyan 渐变背景 + Cyan 边框
- **高级策略卡片**: Gold 渐变背景 + Gold 边框

### 动画
| 效果 | 参数 |
|------|------|

## 10. 交互规格

### Tab 切换
- 点击底部 Tab → 对应内容区 `opacity 1` + `pointer-events auto`
- 其他 Tab 内容区 `opacity 0` + `pointer-events none`
- 切换时自动滚动到顶部
- Active Tab: 图标 Cyan 色 + `scale(1.1)` + 标签 Cyan 色

### 子 Tab/筛选器
- **行情子 Tab**: 热门/涨幅榜/新币/AI概念 — 互斥高亮
- **订单类型 Tab**: 限价/市价/条件 — 互斥高亮
- **K线周期**: 1m/5m/15m/1H/4H/1D — 互斥高亮
- **跟单筛选**: 全部/交易员/AI免费/AI高级/高胜率/稳定收益 — 互斥高亮

### 弹窗交互
- **Nova 对话**: 遮罩点击关闭 / ✕ 按钮关闭
- **订阅弹窗**: 遮罩点击关闭 / CTA 按钮关闭
- **行动卡片**: 点击"一键开仓" → 切换到交易 Tab + 关闭 Nova 弹窗

## 11. 技术规格

### 实现约束
- **单文件 HTML**: CSS/JS 全部内联
- **外部依赖**: 仅 Google Fonts (DM Sans, JetBrains Mono, Noto Sans SC)
- **手机壳**: 390×844px，border-radius 44px，模拟 iPhone 14 Pro
