## 1. 产品定位

**AIEX** 是一个独立品牌的 AI-first 加密货币交易应用。使用 Toobit 底层交易能力（账户体系、撮合引擎、行情数据），但作为独立 App 运营。

### 核心差异化

| 维度 | 传统交易所 (币安/OKX) | 第三方工具 (Cryptohopper) | AIEX |
|------|----------------------|--------------------------|------|
| AI 定位 | 无 / 边缘功能 | 独立于交易所的外挂 | AI 融入交易全流程 |
| 账户体系 | 自有 | 需用户手动配置 API Key | Toobit OAuth 自动授权 |
| 交易延迟 | - | API Key 轮询，秒级延迟 | 内部 API，毫秒级 |
| 跟单 | 有但非 AI 驱动 | 策略市场 | 真人实盘 + AI 策略混合 |
| 价格 | 免费 | $24-107/月 | Free + $9.9-99.9/月 |

### 参考产品

- **RockFlow**: Bobby AI Agent（对话式交易）、跟单系统、低门槛入门
- **Cryptohopper**: 策略市场、分层订阅、自动化交易 Bot

### 目标用户

同时覆盖两类用户：
- **加密新手**: 通过 AI 对话（Nova）降低交易门槛，跟单策略入门
- **有经验交易者**: 通过 AI 辅助分析、自动化工具提升交易效率

## 2. 信息架构

### 4-Tab 结构

```
🧠 智能首页 | 📈 交易 | 🏆 跟单广场 | 👤 我的
```

交易在跟单广场前面——交易是高频操作，需要更容易触达。

AI 不设独立 Tab。AI 助手 **Nova** 以全局浮层形式存在，从任何页面的底部输入框唤起。

## 3. 核心页面设计

### 3.1 智能首页

与传统交易所首页的区别：
- 币安/OKX = 行情列表 + Banner 广告
- RockFlow = Bobby AI 对话为主
- **AIEX = AI 驱动的个性化信息流**

5 个模块（从上到下）：

1. **资产卡片** — 总资产 USDT、今日盈亏百分比、迷你收益曲线
2. **AI 市场快报** — 每日 3 条 Nova 精选洞察，分三类标签：
   - 🔴 风险（红色卡片）：风险预警、需关注的负面信号
   - 🟢 机会（绿色卡片）：看好的机会、积极信号
   - 🟣 洞察（紫色卡片）：市场趋势、链上数据分析
3. **持仓 AI 建议** — 基于用户实际持仓的个性化建议，带一键操作按钮（执行止盈、调整仓位等）
4. **热门策略** — 横滑卡片，AI 策略与真人实盘混排展示
5. **行情概览** — 涨幅榜/跌幅榜/成交量三个子 Tab

底部常驻 Nova 输入框（所有 4 个 Tab 都有）。

### 3.2 交易页

支持现货和合约两种模式，通过顶部 Tab 切换。

#### 通用元素（现货/合约共享）

- **交易对选择器** — 左上角，点击展开币种搜索
- **Nova 自然语言下单栏** — 顶部常驻，支持文字和语音输入。示例："买入 0.1 BTC"、"在 65000 挂限价单"
- **价格显示 + 迷你 K 线** — 当前价、24H高低、成交量、可切换时间周期的简化K线
- **Nova 分析条** — K线下方一行摘要：趋势方向 + 支撑阻力位 + 建议仓位比例，点击展开详情
- **下单面板** — 买/卖切换、订单类型（限价/市价/止盈止损）、价格/数量输入、快捷比例按钮
- **Nova 建议卡片** — 下单面板内嵌，建议止损位、仓位大小，带"自动设置"按钮

#### 合约模式特有

- **方向切换** — 开多/开空
- **保证金模式** — 逐仓/全仓
- **杠杆滑块** — 1x-125x，动态显示风险等级
- **Nova 风控评估面板** — 包含：
  - 风险等级（低/中/高/极高）
  - 预估强平价
  - 建议止损价
  - 建议止盈价
  - "采纳 AI 建议"一键按钮

设计理念：**Nova 不替代传统交易界面，而是在每个关键决策点提供建议**。有经验的用户可以完全忽略 AI 建议，直接用传统方式交易。

### 3.3 跟单广场

三个子 Tab：

#### 实盘交易者

- 筛选栏：综合推荐 / 收益率 / 胜率 / 稳定性 / 跟单人数
- 时间筛选：7天 / 30天 / 90天
- 交易者卡片：
  - 头像 + 昵称 + 等级（金/银/铜牌，基于历史业绩自动评定）
  - 风格标签（合约专家/现货为主、稳健/激进等）
  - 核心数据：收益率、胜率、最大回撤、跟单人数
  - 收益曲线（迷你折线图）
  - 跟单按钮

#### AI 策略专区（付费分层核心场景）

**免费策略**（所有用户可用）：
- 基础网格策略 — 震荡行情适用，低风险
- 定投智选 — Top 10 币种，长期稳健
- 绿色"跟单"按钮直接可用

**高级 AI 策略**（订阅专享，带🔒标签）：
- 趋势追踪 Pro [PRO] — 多时间框架趋势分析 + 链上巨鲸监控
- Alpha 猎手 [ELITE] — 链上早期发现，高风险高回报
- 全自动量化套利 [ELITE] — 跨市场套利，极低风险稳定收益
- 更多高级策略...

高级策略**业绩数据可见但不可跟单**，底部显示金色/紫色"升级解锁"按钮，引导订阅。

**Nova 智能推荐入口** — 页面底部，"不知道跟谁？让 Nova 帮你选"，AI 根据用户风险偏好和资金量推荐最适合的策略组合。

#### 我的跟单

- 已跟单策略/交易者列表
- 实时收益统计
- 可随时取消跟单

#### 分润模式

高水位分润机制：

- 带单者/AI 策略收取利润分成，比例由带单者申请时设定（范围 10-20%，默认 15%）
- **高水位线**：只有当跟单者的累计收益创新高时，才对超出部分计提分成
- **结算周期**：每 7 天结算一次
- **计算示例**：
  - 第 1 周：盈利 $1000，高水位 $1000，分成 = $1000 × 15% = $150
  - 第 2 周：亏损 $300，高水位仍为 $1000，分成 = $0
  - 第 3 周：盈利 $500（累计 $1200），新高超出 $200，分成 = $200 × 15% = $30

### 3.4 我的

- 资产总览（总资产 + 盈亏曲线 + 眼睛隐藏）
- 交易历史
- AI 订阅管理
- Nova 授权等级设置
- 账户绑定（Toobit 账户）
- 设置 / 帮助 / 关于

### 3.5 Nova AI 对话浮层

**Nova** 是 AIEX 的 AI 助手，以全局浮层形式存在。

#### 唤起方式

任何页面底部都有 Nova 输入框，点击后向上滑出半屏对话面板（类似 iPhone 控制中心）。支持向上拖动全屏、向下拖动收起。

#### 上下文感知

Nova 自动获取当前页面信息：

| 唤起页面 | Nova 自动感知 | 快捷操作 |
|---------|-------------|---------|
| 交易页 | 当前交易对、持仓信息 | 下单、设止损、分析币种 |
| 首页 | 全部持仓、市场概况 | 今日热点、持仓分析、推荐策略 |
| 跟单广场 | 用户风险偏好、已跟单 | 帮我选策略、分析交易者、风险评估 |
| 我的 | 总资产、历史交易 | 资产报告、优化持仓、历史复盘 |

#### 行动卡片

Nova 的回复不只是文字。当涉及操作时，返回可交互的行动卡片：
- **下单预览卡片** — 显示方向、数量、价格、止损止盈，一键确认下单
- **策略推荐卡片** — 显示策略详情，一键跟单
- **止盈止损设置卡片** — 滑块调整参数

#### 快捷 Chips

根据上下文动态生成的快捷操作按钮，在输入框下方横滑展示。

#### 授权等级感知

行动卡片根据用户的 Nova 授权等级展示不同操作：
- L1 只读 — 行动卡片只有"查看详情"
- L2 确认执行 — 行动卡片有"确认下单"按钮
- L3 自动执行+通知 — Nova 可自动执行并推送通知
- L4 全自动 — Nova 全自动运行策略

## 4. 订阅体系

### 4 档会员

| 功能 | Free | PRO ($9.9/月) | ELITE ($29.9/月) | MAX ($99.9/月) |
|------|------|---------------|------------------|----------------|
| 免费 AI 策略 | 2 个 | 全部 | 全部 | 全部 |
| 高级 AI 策略 | 🔒 | 3 个 | 全部 | 全部 |
| 同时跟单数 | 1 | 5 | 15 | 无限 |
| 跟单最大金额 | $500 | $5,000 | $50,000 | 无限 |
| Nova 对话次数/日 | 10 | 100 | 无限 | 无限 |
| Nova 自动交易 (L3) | 🔒 | 🔒 | ✓ | ✓ |
| Nova 全自动 (L4) | 🔒 | 🔒 | 🔒 | ✓ (后续开放) |
| 专属 AI 策略定制 | 🔒 | 🔒 | 🔒 | ✓ |

### 定价对标

- Cryptohopper: $24-107/月
- AIEX PRO $9.9/月提供类似 Cryptohopper Explorer($24) 的核心能力，价格优势明显
- 首月 $1 体验价降低试用门槛

### Nova 对话计量

"1 次对话"定义为用户发送 1 条消息并收到 Nova 回复（含行动卡片）。快捷 Chip 点击也计为 1 次。行动卡片上的确认/修改按钮不消耗次数。

### 免费策略说明

Free 用户可使用 2 个指定免费策略（基础网格 + 定投智选）。PRO 及以上用户可使用所有免费策略（随平台新增而扩展）。

### 转化漏斗

1. **免费体验** — 2 个免费策略 + 10 次/日 Nova 对话，让用户体验 AI 价值
2. **看到锁定策略** — 高收益策略可见不可用，历史业绩制造 FOMO
3. **触达付费墙** — 点击高级策略"跟单" → 弹出订阅页
4. **低门槛入门** — PRO $9.9/月，首月 $1
5. **自然升级** — 使用 PRO 后想要更多策略/更大金额 → 升级 ELITE

## 5. 账户体系

### 初期

- Toobit 账户 OAuth 登录
- 登录后自动创建内部 API 授权给 AIEX
- 用户无需手动配置 API Key
- 直接读取 Toobit 账户的资产、行情、交易能力

### 后续扩展（本期不做）

- 支持绑定其他交易所账户（币安、OKX 等）
- 用户登记 API Key + Secret
- 读取对应交易所的资产、行情、交易权限

## 6. Nova 授权等级

渐进式授权，用户自己控制 AI 的权限边界：

| 等级 | 名称 | Nova 能力 | 适合用户 |
|------|------|----------|---------|
| L1 | 只读 | 提供分析和建议，不能执行任何操作 | 新注册用户默认 |
| L2 | 确认执行 | 准备好订单，用户确认后执行 | 建立信任后 |
| L3 | 自动+通知 | 自动执行并推送通知，用户可撤销 | 信任 Nova 的用户 |
| L4 | 全自动 | 完全自动运行策略，定期报告 | 高级订阅用户 |

L3/L4 仅 ELITE 及以上订阅可用。

## 7. 安全红线

1. **不承诺收益** — 任何 AI 建议和策略不保证盈利，必须有明确风险提示
2. **不未授权交易** — 严格按用户设定的授权等级执行，不越级操作
3. **强制风控** — 合约交易必须显示强平价和 Nova 风险评估
4. **数据隔离** — 不同用户的交易数据严格隔离，AI 模型不跨用户学习
5. **资金安全** — 资金始终在 Toobit 账户，AIEX 不托管资金
6. **合规声明** — App 内明确标注风险提示和免责声明
7. **断路器机制** — 当以下任一条件触发时，自动暂停所有 Nova 自动交易（L3/L4）并推送通知：
   - 单币种 5 分钟内涨跌幅超过 ±10%
   - BTC 1 小时内涨跌幅超过 ±8%
   - 用户单日亏损达到账户总资产的 20%

## 8. 新用户引导 (Onboarding)

1. **启动页** — AIEX 品牌 + "AI 驱动的智能交易" slogan

## 12. 技术架构概要

### 前端

- React Native（iOS + Android 双端）
- TradingView Lightweight Charts（K线）
- WebSocket 实时行情推送

### 后端

- Go (Gin) — 核心交易服务、账户服务、跟单服务
- Python — AI 服务（Nova 引擎、策略运行）
- PostgreSQL — 主数据库
- Redis — 缓存、会话、实时数据
- Kafka — 事件流（交易信号、跟单触发）
- ClickHouse — 行情历史数据、分析查询

### AI 引擎

- Claude API — Nova 对话理解、市场分析
- RAG（检索增强生成）— 结合实时市场数据增强 AI 回复
- 自研量化模型 — AI 策略运行
- Pinecone — 向量数据库（用户偏好、历史对话）

### 与 Toobit 集成

- 内部 API（非公开 API Key 方式）— 低延迟
- OAuth 2.0 授权流程
- 资产、行情、交易能力复用 Toobit 基础设施
